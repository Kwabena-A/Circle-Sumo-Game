# 🔵 Circle Sumo Game
A Simple Survival Game That Challenges Your Ability to Evade 
### 🎮 Gameplay 
![Sample Image](Assets/Screenshots/kM2VXnb.png)

## 📦 Features
- Single Player
- Infinite Wave System
- Custom Art
- Dashing
- Weapon Swapping/Spawning (Coming Soon)

## 🕹️ How To Play
- **WASD** To Move
- **Space** To Dash
- **Mouse** + **Left Click** to Attack
